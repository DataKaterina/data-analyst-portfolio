# «Секреты Тёмнолесья» — анализ внутриигровых покупок

Аналитический проект на **SQL (PostgreSQL)**: исследование поведения игроков
MMORPG и их трат на внутриигровую валюту «райские лепестки». Цель — понять,
кто и как покупает в игре, и подготовить выводы для маркетингового отдела.

## Задача

Изучить, как характеристики игроков и их персонажей влияют на покупку
внутриигровой валюты, и оценить активность игроков при совершении покупок.
В частности — проверить гипотезу, что прохождение игры за персонажей разных
рас требует примерно одинакового количества покупок эпических предметов.

## Данные

Схема `fantasy`: игроки (`users`), расы персонажей (`race`), события-покупки
(`events`) и справочник предметов (`items`). Анализируется около 22 тыс.
игроков и 1,3 млн внутриигровых покупок.

Нулевые покупки (`amount = 0`) выявлены отдельно и исключены из расчётов
стоимости, чтобы не искажать средние значения.

## Использованные приёмы SQL

- CTE (`WITH`) с предварительной фильтрацией, чтобы соединения не теряли данные
- Оконные функции (`SUM() OVER ()`) для расчёта долей продаж
- Условные агрегаты через `FILTER (WHERE ...)`
- Подзапрос в знаменателе для корректного расчёта доли уникальных покупателей
- Статистики распределения: `AVG`, `PERCENTILE_DISC` (медиана), `STDDEV`
- `COUNT(DISTINCT ...)` для подсчёта уникальных игроков и покупателей

## Ключевые выводы

- Платит примерно **17,7 %** игроков. По расам разброс заметный: выше всего
  доля у демонов, ниже всего — у эльфов.
- Распределение трат сильно скошено: средний чек значительно выше медианы —
  небольшое число дорогих покупок тянет среднее вверх.
- Две позиции — **Book of Legends** и **Bag of Holding** — дают около
  **97,7 %** всех продаж предметов.
- Покупательская активность **различается по расам** — гипотеза о равенстве
  не подтверждается: расы отличаются и по среднему чеку, и по числу покупок.
